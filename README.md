# 🔐 Gerador de Senhas Seguras

Uma aplicação web moderna, responsiva e ultrassegura para geração de senhas aleatórias personalizadas diretamente no navegador.

## ✨ Recursos Principais
- **Tamanho Customizável:** Escolha o tamanho exato da senha entre 6 e 32 caracteres via slider.
- **Filtros Personalizados:** Opções para incluir/excluir maiúsculas, minúsculas, números e símbolos especiais.
- **Aleatoriedade Criptográfica:** Utiliza a API `window.crypto` nativa dos navegadores modernos para garantir senhas imprevisíveis.
- **Indicador de Força em Tempo Real:** Feedback visual instantâneo que classifica a senha como Fraca, Média, Forte ou Muito Forte.
- **Cópia Rápida:** Botão para copiar a senha para a área de transferência com um único clique e feedback visual.
- **Geração Instantânea:** Atualização automática ao alterar qualquer parâmetro ou ao clicar no botão de regeração.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Marcação semântica e acessível.
- **CSS3:** Design responsivo, variáveis CSS, layout em Card e tema escuro moderno.
- **JavaScript (Vanilla JS):** Lógica pura sem dependências externas, utilizando APIs nativas (`Crypto` e `Clipboard`).

## 🚀 Como Usar
1. Baixe ou clone este repositório em sua máquina local.
2. Certifique-se de que os arquivos `index.html`, `styles.css` e `script.js` estão na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador web moderno (Chrome, Firefox, Edge, Safari).
4. Ajuste o tamanho da senha e selecione as categorias de caracteres desejadas.
5. Clique em **Copiar** para usar sua nova senha com segurança!
